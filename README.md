# NAMU DESIGN 官网

这是 NAMU DESIGN 呐姆设计新版官网的静态版本，第一版以“体验设计 / 产品体验 / 高保真交互原型”为主线，只上线方太和 LV 两个深度案例。

## 页面

- `/` 首页
- `/cases/fotile.html` 方太小程序首页改版
- `/cases/lv.html` LV 数字体验内容整理
- `/sitemap.xml`
- `/robots.txt`
- `/llms.txt`

## 部署

默认部署到 GitHub Pages：

1. 将仓库推送到 GitHub。
2. 在仓库 Settings → Pages 中选择 `main` 分支根目录。
3. 保留根目录 `CNAME`，域名为 `www.namu.design`。
4. DNS 中将 `www.namu.design` 指向 GitHub Pages。

## 上线前确认

- 方太、LV 客户名称和项目内容是否可公开。
- `assets/fotile-figma.png`、`assets/lv-figma.png` 是否为最终可公开配图。
- ICP 备案号是否需要在页脚展示。
- 如 LV 有独立 Figma 源文件链接，可替换当前从 INTRO 导出的素材总览图。
